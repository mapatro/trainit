Every chapter has the same rhythm: a tight idea → a ● Do this → an AI prompt you paste into Claude/ChatGPT/Gemini to produce a real ClearEaty asset → a ✓ Check yourself. Do them in order — each builds the growth engine.

1

Anatomy of a converting page

A high-converting app page is a sequence: a clear promise, how it works in 3 steps, what you get, proof, FAQ, and a final install button. One message, repeated.

2

Write the hero

The hero is 90% of the battle: one promise, one subline, one button.

ai prompt · hero copy
› Write 5 hero options for the ClearEaty landing page. Each = a headline (≤8 words), a one-sentence subline, and a button label. ClearEaty: free app, scan a barcode or snap an ingredients label → honest 1–10 health score, every ingredient explained, healthier picks; no ads, no sign-up; US. Audience: health-curious shoppers + South Asian diaspora. No medical claims. Make them visual and confident.
3

How it works + benefits + proof

ai prompt · page sections
› Write the rest of the ClearEaty landing page, section by section: (1) “How it works” in exactly 3 steps (Scan → See the score → Find a healthier pick); (2) 4 benefit blocks (title + 1 sentence each); (3) a social-proof section format (ratings/quotes placeholders); (4) 6 FAQ items including “Is this medical advice?” (answer: no, informational only) and “What does it cost?” (free, no ads); (5) a final call-to-action block. Honest voice, no medical claims.
● Do this
  1. Paste the App Store / Play badges and point them at the live listings (US).
  2. Make the install button the only primary action on the page.
4

Ship it

ClearEaty already owns cleareaty.com. Publish the page there (or as a fast static page) and make sure it loads quickly on a phone.

● Do this
  1. Hand the copy to whoever owns the site, or build a simple static page (same approach as this TrainIt site — HTML + GitHub Pages).
  2. Test on a real phone: does it load fast and is the install button reachable in one thumb tap?
✓ Check yourself
  • Every section written and on-brand
  • Exactly one primary CTA (install)
  • “Not medical advice” appears in the FAQ
  • Loads fast and looks right on mobile

More: web.dev — fast, mobile-friendly pages

Remember: ClearEaty is an information tool, not medical advice. Never write “cures,” “treats,” “detox,” or “guaranteed weight loss.” Keep it about knowing what’s in your food.
