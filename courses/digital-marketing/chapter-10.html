1

How ad accounts are structured

Every ad platform shares one shape: Campaign (the goal) → Ad set (who you target + budget) → Ad (the creative). You test by running a few ads per audience and keeping winners.

2

Start with the cheapest honest test

Pick one channel (often Meta/Instagram for this audience), a small daily budget, and goal = app installs. Because the app is free, measure cost per install and — crucially — cost per activated user (install that scans). ROI shows up later via retention/referral, so don’t over-spend before organic proves the message.

● Do this
  1. Set a small daily test budget you’re comfortable losing while learning.
  2. Define success up front: target cost-per-install and a minimum first-scan rate.
  3. Run for long enough to exit the learning phase before judging.

No paid subscription yet = no direct ad revenue to recoup. Treat paid as buying learnings + early users, not ROI-positive day one. Tools, prices and ad policies change fast — verify the current name, pricing, and rules before you rely on them. Never lock the work into one paid tool.

3

Facebook & Instagram ads

ai prompt · meta ad set
› Create a Meta (Facebook + Instagram) app-install ad plan for ClearEaty (free food-label scanner; US; also South-Asian-diaspora interest). Give: 2 audience ideas (e.g. health-curious parents; diaspora food interests) with targeting suggestions, 4 primary-text variants, 4 headlines, and 3 creative concepts (incl. the “scan a snack → score” video). Honest, no medical claims. Add a simple test plan: budget split, what to measure, and when to kill vs scale.
4

Reddit ads

Reddit ads let you target by subreddit and interest — perfect for niche, high-intent health/food communities. Native, non-salesy creative wins; redditors smell ads instantly.

ai prompt · reddit ads
› Plan a Reddit Ads test for ClearEaty. Suggest 8 relevant subreddits/interests to target (nutrition, healthy eating, frugal, parenting, South-Asian communities). Write 4 ad variants that feel native to Reddit (curious, honest, low-hype), each with title + body + CTA. Note Reddit ad-policy points to verify for a health-adjacent app, and a small test budget plan.
● Do this
  1. Launch one Meta test and one Reddit test with the same offer; compare cost per activated user.
5

Google & Apple search

Capture high-intent searchers: Google App campaigns (Play + search/YouTube) and Apple Search Ads (App Store). Bid on your brand + category terms like “food scanner” / “ingredient checker.”

● Do this
  1. Set up a small Apple Search Ads campaign on brand + 3–5 category keywords.
  2. Set up a Google App campaign with your best text + the scan video; let it optimise to installs.
✓ Check yourself
  • One Meta + one Reddit test live with clear success metrics
  • Apple Search + Google App campaigns set up
  • You measure cost per activated user, not just installs
  • A weekly kill/scale decision rule written down

More: Meta Ads policies & setup
