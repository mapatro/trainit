Every chapter has the same rhythm: a tight idea → a ● Do this → an AI prompt you paste into Claude/ChatGPT/Gemini to produce a real ClearEaty asset → a ✓ Check yourself. Do them in order — each builds the growth engine.

1

How search intent works

SEO = matching what people search with a page that answers it. For ClearEaty the goldmine is ingredient/additive questions and “healthiest [category]” comparisons — high intent, naturally on-brand.

2

Build keyword clusters

ai prompt · keyword clusters
› Build SEO keyword clusters for ClearEaty (free food-label scanner). Create 5 topic clusters (e.g. additives, reading labels, healthiest-category comparisons, kids’ snacks, diaspora/South-Asian groceries). For each cluster list 8–12 long-tail keywords as real search questions, mark intent (informational/commercial), and suggest one “pillar” page + supporting articles. Flag anything to verify in a keyword tool.
3

Outline + draft an article

ai prompt · article draft
› Write a helpful, accurate, non-medical article for ClearEaty’s blog titled “How to actually read a packaged-food label (in 60 seconds).” Structure: intro, the 4 things to check, common confusing additives explained simply, a quick checklist, and a soft CTA to scan products with ClearEaty for an instant score. Plain language, cite reputable sources (FDA/WHO/Open Food Facts), no health claims. ~900 words with H2/H3 headings.
● Do this
  1. Fact-check every claim against a reputable source before publishing.
  2. Add internal links: article → pillar page → “get the app.”
4

On-page SEO + publish

● Do this
  1. Set a clear title tag + meta description with the main keyword.
  2. Add the install CTA mid-article and at the end; link related articles.
  3. Submit the page in Google Search Console and watch impressions.
✓ Check yourself
  • 5 keyword clusters saved
  • 1 article fact-checked and published
  • Internal links point to the install CTA
  • Page indexed in Search Console

Helpful + accurate beats keyword-stuffed. Remember: ClearEaty is an information tool, not medical advice. Never write “cures,” “treats,” “detox,” or “guaranteed weight loss.” Keep it about knowing what’s in your food. Tools, prices and ad policies change fast — verify the current name, pricing, and rules before you rely on them. Never lock the work into one paid tool.
